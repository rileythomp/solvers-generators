This site is a collection of solvers and generators I've written in Go and JavaScript. So far it has 3 sections, mazes, Sudoku, and Truchet tiles.

Mazes

Jamis Buck has a blog series from 2011 that I came across which contains descriptions and animations of 11 maze algorithms. Here is the list of all 11 maze generation algorithms that I've implemented in Go. You can see more details about each algorithm under the Mazes section.

  • Recursive Backtracking
  • Eller's Algorithm
  • Kruskal's Algorithm
  • Prim's Algorithm
  • Recursive Division
  • Aldous-Broder Alogirthm
  • Wilson's Algorithm
  • Hunt and Kill Algorithm
  • Growing Tree Algorithm
  • Binary Tree Algorithm
  • Sidewinder Algorithm
The source code for all the maze generation algorithms can be found here.
